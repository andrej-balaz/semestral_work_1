# Statistical Data Analysis Project

## Overview
This project presents a set of statistical analyses performed on simulated coursework datasets as part of a university statistics assignment. The goal was to practice and demonstrate the application of statistical methods in Python and interpret results in a clear and structured way.

The project focuses on hypothesis testing, confidence interval estimation, correlation analysis, group comparisons, and ANOVA-based methods.

---

## Key Skills Demonstrated
- Data cleaning and preprocessing
- Exploratory data analysis
- Statistical hypothesis testing
- Confidence interval estimation
- One-way ANOVA and post-hoc analysis (Tukey HSD)
- Data visualization
- Working with structured datasets in Python

## Analyses Performed

### 1. Car Sales Analysis
- Analysis of simulated vehicle sales data (combustion, hybrid, electric)
- Confidence interval estimation for electric vehicle proportion
- Hypothesis testing of changes between 2024 and 2025

---

### 2. Temperature vs Beverage Sales
- Analysis of simulated relationship between temperature and hot beverage revenue
- Tests for independence and correlation

---

### 3. Physical Proportions Study
- Comparison of arm span and body height
- Hypothesis testing for equality of means

---

### 4. Engine Performance Decline
- Comparison of simulated performance loss between two manufacturers
- Normality testing and confidence interval estimation
- Statistical comparison between groups

---

### 5. Tipping Behavior in Pubs
- Analysis of simulated tipping data across multiple pubs
- One-way ANOVA to test equality of means
- Tukey HSD post-hoc test to identify specific differences between groups
